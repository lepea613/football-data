# Bundesliga Data Analysis Project

This project collects, stores and analyzes Bundesliga football data.

## Features
- Fetch data from API
- Store in SQL database
- Analyze matches
- Visualize top teams

## Technologies
- Python
- Pandas
- SQLite
- Matplotlib

## How to run

```bash
pip install -r requirements.txt
python fetch_data.py
python database.py
python analysis.py
